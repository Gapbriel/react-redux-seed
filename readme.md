## Usage

### Installation

```
$ sudo npm install
```

### Run Unit Testing

```
$ npm test
```

### Run App

Create <b>bundle.js</b> in public folder by the command:
```
$ webpack -w
```

Then. open <b>public/index.html</b> file in any browser.
